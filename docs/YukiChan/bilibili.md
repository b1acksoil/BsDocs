## 指令帮助

### #bilibili av - 通过 AV 号获取视频信息

**指令格式**

```:no-line-numbers
#bilibili av <AV号>
```

**使用示例**

```:no-line-numbers
#bilibili av 170001
```

### #bilibili bv - 通过 BV 号获取视频信息

**指令格式**

```:no-line-numbers
#bilibili bv <BV号>
```
