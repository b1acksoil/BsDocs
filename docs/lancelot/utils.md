# 实用工具

## 指令帮助

指令格式中，中括号 `[]` 中的内容为**可选参数**，尖括号 `<>` 中的内容为**必选参数**。

### /mainbot - 切换群组内主 Bot

**指令格式**

```:no-line-numbers
@SpecificBot /mainbot
```

**使用示例**

```:no-line-numbers
@Bedivere /mainbot
```

**备注**
注意手动 @ 与复制的 @ 实际并不一样，所以复制的是无效的。


### /choose - 随机选择

**指令格式**

```:no-line-numbers
/choose <选项1> <选项2> [选项3] [选项4] ...
```

**使用示例**

```:no-line-numbers
/choose 推分 摆烂
```
