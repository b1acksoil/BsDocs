# 简介

lancelot.bot 是一个跨平台的聊天机器人 (Bot)，基于 Node.js + TypeScript 开发，框架使用 [Koishi.js](https://koishi.js.org) ([repo](https://github.com/koishijs/koishi))。

Bot 目前主要提供音游 Arcaea 的查分服务，未来有可能发展出更多功能。
